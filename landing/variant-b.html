MENTIONWATCH(1) MentionWatch Manual MENTIONWATCH(1)
NAME
mentionwatch — social media keyword monitoring API
FREQUENTLY ASKED QUESTIONS
How fast are mentions delivered?
We poll sources every 5–10 minutes. Most mentions arrive within 15 minutes of being posted. Priority plan customers get faster polling intervals.
What happens if my webhook endpoint is down?
We retry 3 times with exponential backoff: 10s, 60s, 5 min. Failed deliveries are stored and retrievable via GET /api/mentions so nothing is lost.
How do I verify webhooks are from MentionWatch?
Every delivery includes an X-MentionWatch-Signature header containing an HMAC-SHA256 of the request body signed with your webhook secret. Node.js and Python examples are in the verification docs.
Which sources are currently live?
Hacker News and Reddit are live. Twitter/X, LinkedIn, Product Hunt, Dev.to, and GitHub are in progress. Vote for sources at hello@mentionwatch.com.
Can I cancel anytime?
Yes. Cancel from your Stripe billing portal. No cancellation fee. You keep access until the end of the billing period.
Why is this cheaper than Mention.com or Brand24?
We build an API, not a dashboard product. No UI team, no analytics infrastructure, no enterprise sales layer. Less overhead means lower price. You write the UI if you want one.
