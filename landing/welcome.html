Quick Start (90 seconds)

Step 1: Add a keyword
curl -X POST https://api.mentionwatch.com/api/keywords \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"keyword": "my-product"}'
Step 2: Set your webhook URL
curl -X POST https://api.mentionwatch.com/api/webhooks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://your-app.com/webhook"}'
Step 3: Send a test webhook
curl -X POST https://api.mentionwatch.com/api/webhooks/test \
  -H "Authorization: Bearer YOUR_API_KEY"

This sends a sample payload to your webhook URL immediately.
