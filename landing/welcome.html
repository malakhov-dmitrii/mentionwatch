Quickstart — 90 seconds

1

Add a keyword to monitor

POST a keyword string. MentionWatch begins scanning immediately across all active sources.

01-add-keyword.sh 
$ curl -X POST https://api.mentionwatch.com/api/keywords \
    -H "Authorization: Bearer $MW_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"keyword": "my-product"}'
2

Register your webhook endpoint

Point MentionWatch at your server. Every keyword match fires a signed POST to this URL.

02-set-webhook.sh 
$ curl -X POST https://api.mentionwatch.com/api/webhooks \
    -H "Authorization: Bearer $MW_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"url": "https://yourapp.com/webhooks/mentions"}'
3

Fire a test payload

Sends a sample mention event to your webhook so you can verify your handler before real mentions arrive.

03-test-webhook.sh 
$ curl -X POST https://api.mentionwatch.com/api/webhooks/test \
    -H "Authorization: Bearer $MW_API_KEY"

# Your endpoint receives:
{
  "event":    "mention.created",
  "keyword":  "my-product",
  "source":   "hackernews",
  "title":    "Show HN: My Product is live",
  "url":      "https://news.ycombinator.com/item?id=..."
}
